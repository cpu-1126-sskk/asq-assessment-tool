# ASQ Assessment Tool (情感风格问卷)

![ASQ Preview](https://img.shields.io/badge/Status-Beta-blueviolet?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Stack-HTML--CSS--JS-blue?style=for-the-badge)

这是一个针对 **情感风格问卷 (ASQ)** 的交互式测评网页应用。基于 Hofmann & Kashdan (2010) 的研究理论，通过 20 道题目深度评估个体在三个维度的情绪调节风格：

1.  **隐藏 (Concealing)**：情绪内部化倾向。
2.  **调整 (Adjusting)**：情绪管理与适应能力。
3.  **容忍 (Tolerating)**：对负面情绪的接纳度。

## ✨ 特性 (Features)

-   **Premium UI**: 采用深色玻璃拟态 (Glassmorphism) 设计，提供沉浸式测评体验。
-   **Interactive Logic**: 题目焦点跳转与进度实时反馈。
-   **Data Visualization**: 使用 Chart.js 动态生成多维雷达图。
-   **Responsive Design**: 完美适配移动端与桌面端。

## 🚀 快速开始 (Quick Start)

1.  克隆本项目：
    ```bash
    git clone https://github.com/cpu-1126-sskk/asq-assessment-tool.git
    ```
2.  直接双击 `index.html` 即可在浏览器中运行。

## 📊 计分说明 (Scoring)

-   1-5 分制评分。
-   包含 1 道反向计分题 (Item 19)。
-   结果以各维度均分形式呈现。

*Note: 本工具仅供自我探索使用。测评结果受个体主观感受影响，不具备临床诊断效能。*
